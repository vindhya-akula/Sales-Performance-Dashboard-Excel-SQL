# Sales-Performance-Dashboard-Excel-SQL
Interactive Sales Performance Dashboard built using Excel Pivot Tables, Pivot Charts, KPI Cards, Slicers, and SQL-based business analysis.

## Project Objective
The objective of this project is to analyze sales performance across products, regions, customers, and sales representatives using Microsoft Excel and SQL. The dashboard provides interactive insights into revenue, profit, order volume, and sales trends to support data-driven business decisions.

--->The project demonstrates the use of:
Excel Pivot Tables
Pivot Charts
KPI Cards
Slicers and Timelines
Data Visualization
SQL Business Analysis

## Dataset Used
A sales transaction dataset containing 1,000 records from 2024–2026. The dataset represents sales transactions across multiple products, categories, regions, and sales representatives.

## Business Questions & KPIs
## KPI Metrics
1. Total Revenue
What is the total revenue generated?
2. Total Profit
What is the total profit earned?
3. Total Orders
How many orders were placed?
4. Total Quantity Sold
How many units were sold?
5. Average Order Value
What is the average revenue generated per order?
6. Profit Margin 
What percentage of revenue was converted into profit?
